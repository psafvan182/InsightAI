# InsightAI

## 📌 Project Overview

InsightAI is an AI-powered Data Analysis Assistant built using Python, Streamlit, Pandas, NumPy, Plotly, and Scikit-learn.

It helps business owners upload datasets, perform automatic analysis, visualize insights, and interact with their data using AI.

## 🚀 Features

- Upload CSV or Excel files
- Automatic data cleaning
- Exploratory Data Analysis (EDA)
- Interactive charts
- Machine Learning predictions
- AI chatbot for data queries

## 🛠 Technologies

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- OpenPyXL
